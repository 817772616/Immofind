# Immofind
Crawl your favorite place to live padre!

**Installation**  
``` 
python3 setup.py
sudo apt install chromium
```


**Usage**  

1. Automated setup
```
crontab -e
* */6 * * * python3 crawl.py >/dev/null 2>&1
python3 app.py
```

3. Manual
```
python3 crawl.py
python3 app.py
```
